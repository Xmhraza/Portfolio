This is a simple java game (14 scripts) using Object Oriented Programming

This file only has the java files. If you want to run the game, go to the mainProgram file and run the main method. 

The game is similar to tower defense where there will be a set of waves before you reach the final boss.

Functionality:

1. The game has a tutorial where it introduces all the mobs and is text based
2. Multiple mobs (made using inheritance of a common class)
3. A boss mob with different abilities
4. A randomly generated enviroment that can benefit or go against you
5. A shop system where you can increase your stats

Please note, the game is text based and does not have any UI involved
